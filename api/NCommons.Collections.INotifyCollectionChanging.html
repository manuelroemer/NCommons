Interface INotifyCollectionChanging

Notifies listeners about when a collection is about to change, but before the change is actually going to happen.

Namespace: NCommons.Collections
Assembly: NCommons.Collections.dll
Syntax
public interface INotifyCollectionChanging

Remarks

This interface can be treated as the counterpart of the INotifyCollectionChanged interface and should be implemented together with it.

The PreviewingObservableCollection<T> is a finished collection which implements both of these interfaces.
