Class Optional

Defines static utility functions and extension methods which assist with the creation of and interaction with Optional<T> instances.

FlatMapNonNull<T, TResult>(Optional<T>, Func<T, Optional<TResult>>)

If the optional holds a value which is not null, the specified flatMapValue function is invoked with the optional's held value. The returned value is then returned by this method.

If the optional is either empty or holds a value which is not null, the flatMapValue function is not invoked. Instead, an empty Optional<T> instance is returned.

    where T : class

Parameters

Optional<T> optional

The optional which may hold a nullable value.

Func<T, Optional<TResult>> flatMapValue

The function to be invoked if the optional holds a value which is not null. This function receives the optional's held value as a parameter.

Returns

Optional<TResult>

The Optional<T> instance returned by the flatMapValue function if the optional holds a value which is not null or an empty Optional<T> instance if the optional is empty.

MapNonNull<T, TResult>(Optional<T>, Func<T, TResult>)

If the optional holds a value which is not null, the specified mapValue function is invoked with the optional's held value. The returned value is then wrapped in a new Optional<T> instance and returned by this method.

If the optional is either empty or holds a value which is not null, the mapValue function is not invoked. Instead, an empty Optional<T> instance is returned.
