ToEmptyIfNull<T>(Optional<T>)

Returns an empty optional if the current optional's held value is null. Otherwise, returns a non-empty optional holding the same value as the current optional.

Declaration

public static Optional<T> ToEmptyIfNull<T>(this Optional<T> optional)
    where T : class

Parameters

Optional<T> optional

The optional holding a nullable value.

Returns

Optional<T>

An empty optional if the current optional's held value is null. Otherwise, a non-empty optional holding the same value as the current optional.

Type Parameters

T

The type which may be held by the optional.

Remarks

Recommendation: This method is ideally used together with C# 8.0's nullable reference types, because it strips the nullable type annotation from a nullable reference type, making it obvious that a held value can no longer be null.
