Namespace NCommons.Collections

Classes

BinaryHeap<T>

Stores and prioritizes elements based on a max binary heap. The contained elements are prioritized by an IComparer<T>.

Heap<T>

An abstract base class for any PriorityQueue<T> implementation that uses a heap of any kind for managing the data.

NotifyCollectionChangingEventHandler

Represents the method that handles the CollectionChanging event.

PreviewingObservableCollection<T>

A dynamic data collection which provides notifications about when items are going to change and when they have actually changed.

PriorityQueue<T>

Stores and serves items ordered by a priority.

WeakReferenceCollection<T>

A special collection which only holds weak references to the elements that are added to it. This allows the elements that are added to this list to be garbage-collected.

Interfaces

INotifyCollectionChanging

Notifies listeners about when a collection is about to change, but before the change is actually going to happen.

IPriorityQueue<T>

Represents a data structure which stores and serves items ordered by a priority.
