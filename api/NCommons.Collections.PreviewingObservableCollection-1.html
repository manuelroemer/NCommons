IDisposable

An IDisposable which is used to scope the lifetime of the reentrancy block.

Remarks

The only exception to this rule is if CollectionChanging has 1 or less subscribers.

This method has the same functionality as BlockReentrancy(), but only for the CollectionChanging event.

Deriving classes must call CheckCollectionChangingReentrancy() for throwing the exception.
